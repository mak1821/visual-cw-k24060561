Introduction

This project explores Olympic medal data in relation to GDP, aiming to uncover patterns in country performance over time. It uses data from the Kaggle, World Bank, and the Maddison Project, combined with D3.js for an interactive visualisation.

Implementation Overview

This visualisation investigates how Olympic medal success relates to a country’s economic scale. It presents an interactive choropleth map showing a log-scaled Medal Performance Score (MPS) for each country per Olympic year. The aim is to highlight countries that perform exceptionally relative to their size and resources, and those that underperform.

Data prior to 1950 was excluded due to incomplete GDP coverage, especially for developing and transitional economies. This ensures reliable comparisons based on consistently available economic indicators.

Data Preprocessing and Cleaning

The original Olympic dataset contained inconsistencies in country representation. Country names were primarily listed under the "Team" field, and NOC codes did not always follow ISO standards. To resolve this:

  • Country names were first matched against entries in the world cities dataset.
  • If a match failed, a custom alias mapping was used.

This process produced a cleaned dataset with consistent ISO codes, forming the basis of the analysis and visualisation. Transitional regions like the USSR and East Germany were retained if GDP data was available.

Dataset Sources

  • Olympic Medal Data: Olympics dataset
  • ISO Matching: World cities dataset
  • GDP Data: Merged from World Bank and
  • GDP Conversion: World Bank GDP (2017 USD) was converted to 2011 USD using a PPP factor of 0.533 (PPP Conversion Factor)

All participating countries were retained in the dataset even if they earned zero medals in a given year. This ensured the visualisation reflected both participation and performance.

Medal Performance Score Formula

MPS = Total Medals ÷ GDP per Capita

  • This score adjusts medal counts based on a country's economic scale, providing a normalized view of Olympic performance.
  • To compress extreme values and support choropleth encoding, the final score was transformed using a base-10 logarithm.
